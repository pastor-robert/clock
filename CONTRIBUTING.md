# Contributing

Contributions are welcome.

## Development Setup

1. Clone the repository
2. Run `nix develop` to enter the development environment
3. Run `./scripts/setup-hooks.sh` to install the pre-commit hook
4. Run `npm install` to install linting dependencies

## Code Style

- Run `npm test` before committing to check linting
- The pre-commit hook will validate HTML, CSS, JS, and JSON files

## Pull Requests

- Keep changes focused and minimal
- Ensure all CI checks pass
